# 🎓 KCT Admission Portal

A responsive and modern **College Admission Portal** built using **HTML5** and **CSS3**. This project is designed to showcase a complete admission portal with an attractive user interface and multiple sections commonly found on college websites.


## 📌 Features

- 🏫 Responsive Home Page
- 📖 About College Section
- 📚 Courses Offered
- 📝 Online Admission Form
- 📊 Placement Statistics
- 🏆 College Achievements
- ⭐ Student Reviews
- 🖼️ Image Gallery
- 🎥 Campus Tour Video
- 🎵 College Anthem Audio
- 📍 Google Maps Integration
- ❓ Frequently Asked Questions (FAQ)
- 📞 Contact Information
- 📱 Responsive Design
- 🎨 Modern UI using HTML & CSS



## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Maps Embed

## 🚀 How to Run the Project

1. Download or clone this repository.

bash
git clone https://github.com/shreenidhi13-ai/KCT-Admission-Portal.git


2. Open the project folder in **Visual Studio Code**.

3. Install the **Live Server** extension.

4. Right-click **index.html**.

5. Click **Open with Live Server**.

The website will open in your default browser.
